003 : PJMAPP

"PJMAPP" is a polish sign language learning app that enables users to learn basic gestures and expressions of sign language. In addition to interactive sign language lessons, the app also offers quiz learning functionality to test the user's knowledge.

Tools Used :

  • Frontend: Expo, React Native
  • Backend: Firebase

Features :

  • Interactive sign language lessons.
  • Quiz learning to test knowledge of gestures.
  • Dictionary of sign language gestures with descriptions.
